# L-Hub
mc-hub rewrite

compile with maven

features:
- 
- cancel all build/interact events
- inventory items to join server
- server commands
- server portals
- /gamemode command
- Clear weather
- Custom chat
- Custom Join/Leave messages
- Scoreboard
- Staffmode with vanish
- /stop command which nicely kicks all players
- /playtime command
- tp to spawn on join with /setspawn command
